Release Notes

What's new in Atomic. We ship continuously — this is the human-readable log.

Live version —
1.2.0LatestJul 6, 2026

Live cross-chain swaps on atomicpay.cloud — real routing, analytics, and a full light redesign.

  • Live swaps via LI.FI — gatekeeper-free BTC + EVM cross-chain routing with a fail-closed asset map. Net platform fee 2.5% (customer all-in ~2.75%).
  • In-wallet execution — your connected wallet signs the route directly; EVM approval + chain-switch handled, with Bitcoin and Solana paths.
  • Wallet connect overhaul — EIP-6963 multi-wallet discovery, in-modal WalletConnect QR, 16 wallets incl. Bitcoin, connect spinner + timeout, and a guard that blocks swaps when your wallet can't send the source asset.
  • Human-readable amounts — type "10", not "10000000".
  • New light design across landing, swap, transfers, and checkout.
  • Public transfers explorer — every conversion, filterable and paginated.
  • Daily P&L email — today / week / month / YTD, 7 AM ET.
  • Analytics + Help — PostHog product analytics and session replay, plus this Help & bug tracker.
1.1.0Jul 3, 2026

Build metadata, guarded broadcast adapters, and launch-readiness evidence.

  • Runtime build metadata exposed via /v1/build, /v1/health, and project progress.
  • Guarded wallet broadcast adapters for EVM and Solana (simulation / live / live-with-fallback).
  • Swap broadcast route with transaction-proof capture and raw-transaction redaction.
  • Operator audit + settlement reconciliation evidence exports with SHA-256 digests.
  • Production readiness gates and expanded operator-protected smoke coverage.

Initial Atomic Payments foundation.

  • First MVP foundation — payment intents, merchant + wallet models, and the core API.
